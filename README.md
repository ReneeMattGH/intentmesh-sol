# IntentMesh

AI-powered intent execution for Solana. Describe what you want to do in plain English, and let the AI analyze real-time market data to execute your trades.

## Features

- **Natural Language Trading**: Say what you want, AI handles the rest
- **Real-Time Market Data**: Live prices from Jupiter and CoinGecko APIs
- **Wallet Integration**: Connect your Solana wallet for seamless transactions
- **Risk Analysis**: Built-in safety checks before execution
- **Transaction History**: Track all your trades and intents

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Solana Web3.js
- Jupiter API
- Radix UI

## Getting Started

```bash
npm install
npm run dev
```

## Building for Production

```bash
npm run build
```
